# Project Ideas and Research Notes

---

## Current Goal

Build a Raspberry Pi based autonomous robot using computer vision and real-time control.

---

## Future Features

### vision 
- Tracking red objects using OpenCV.
- Find the centre of the line.
- Obstacle detection.
- recognise different shapes and colours.

### navigation
- PID steering control for line following.
- Path planning and obstacle avoidance.
- Autonomous navigation.

### Hardware
- Raspberry Pi 5.
- Camera module 3.
- Motor driver board.
- Ultrasonic distance sensor.
- Battery pack.
